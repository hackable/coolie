Coolie

Coolie is deployment tool based on rysnc lets you ship apps to remote machines and vise versa using just one command.

View project onGitHub

Installing

Go to your project root.

cd your_project

git clone git://github.com/hackable/coolie.git

cd coolie

chmod +x coolie.sh

Configuration.

Go to config file in coolie and change options according to your needs.

SSH_USER - The username on remote host SSH_HOST - The hostname on remote host SSH_KEY - The IdentityFile on remote host LOCAL_DIR - The directory of your project on local machine REMOTE_DIR= The directory of your project on remote machine

Usage

./coolie.sh --local

This will sync and download remote files to your local host.

./coolie.sh --remote

This will sync and upload local files to your remote host.
